# Tarea-2-Datos-II

Para poder ejecutar este programa es necesario compilaro primero para esto se ingresa el siguiente comando en consola g++ -o times times.cpp `pkg-config --cflags --libs gtk+-3.0`, el apartado de flags se encarga de econtrar la ruta a la libreria gtk, esto es necesario ya que esta libreria se necesita para poder graficar los resultados, una vez se haya compilado el arhivo se procede a ejecutarlo para esto se ejecuta el siguiente comando, ./times, para este caso el nombre del ejecutable es "times", en el caso de nuestro ejecutable tenga un nombre diferente se colocaria de la siguiente forma ./nombre_del_ejecutable. Si se desea modificar los datos de prueba es necesario ir a la linea de codigo n√∫mero 17 y cambiar los valores que se encuentran en el vector dataSizes, una vez se hayan cambiado estos valores se debera volver a compilar el archivo y ejecutarlo.
